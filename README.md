# Monster and Memories — Party Planner

An unofficial fan-made party planning tool for the Monsters and Memories MMO.

🔗 Live site: https://aiyasei.github.io/mnm-party-planner/

## Features
- Build a party (up to 6 classes) and see capability coverage at a glance
- Identify gaps in Crowd Control, Debuffs, Healing, Buffs, and Utility
- See which abilities overlap between your chosen classes
- Full searchable/filterable spell library for all 18 classes
